Algorithms Animated

I'm currently working on a project named "Algorithms Animated" which is a web based educational tool that students can use to see a visual representation of certain algorithms. The idea was sparked when I was looking for a visualization of the "Max Independent Set" algorithm for my Algorithms course in my Senior year at Rutgers, but couldn't find one on the internet.
